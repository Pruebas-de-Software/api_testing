# api_testing
Repositorio del curso enfocado en testing de APIs e integración. Incluye diseño, implementación y validación de una API de tareas con Swagger, API Key y pruebas automatizadas. Busca desarrollar habilidades prácticas en QA moderno, cubriendo pruebas funcionales, negativas, de contrato y seguridad.
